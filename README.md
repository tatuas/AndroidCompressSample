# AndroidCompressSample

複数ファイルを、tar形式、zip形式で圧縮するサンプルです。
圧縮ファイルは``/sdcard/``直下に吐き出されます。

# 使用前に
``/sdcard/``直下に
```
1.jpg
2.jpg
3.jpg
4.jpg
5.jpg
```
という名前のファイルを用意してください。

# Marshmallow以降の場合
アプリ設定から権限を許可して下さい。
